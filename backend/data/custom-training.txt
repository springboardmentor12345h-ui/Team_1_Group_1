## About CampusEventHub

CampusEventHub is an inter-college event management platform that connects students and colleges for various academic, cultural, technical, and sports events.

The platform supports:

- Event Discovery
- Online Event Registration
- Role-Based Access (Student / College Admin)
- Event Creation & Management (Admin Only)
- Participant Approval System
- Registration Tracking
- Feedback Submission
- Dashboard Overview
- Multi-College Event Support

CampusEventHub operates as a secure platform with authentication and role-based access control.

## STUDENT GUIDANCE

### How to Create an Account:

1. Select the option to register on the platform.
2. Choose the role "Student".
3. Enter required details (Name, Email, College, Phone).
4. Create a secure password.
5. Complete the registration process.
6. Login using your credentials.

---

### How to Login:

1. Navigate to the login section.
2. Enter your registered email and password.
3. Click Sign In.
4. Access your student dashboard.

---

### How to Browse Events:

1. Login to your account.
2. Go to the Events section.
3. View available events.
4. Use filters (Event Type, Date, College) if available.
5. Select an event to view details.

---

### How to Register for an Event:

1. Login to your student account.
2. Browse available events.
3. Select the event you wish to join.
4. Review event details and eligibility.
5. Click the Register option.
6. Confirm your registration.
7. Track your status in "My Registrations".

---

### How to Track Registrations:

1. Login to your account.
2. Open your dashboard.
3. Navigate to "My Registrations".
4. View status (Pending / Approved / Rejected).

---

### How to Submit Feedback:

1. After attending an event, open "My Registrations".
2. Select the completed event.
3. Choose the feedback option.
4. Provide ratings and comments.
5. Submit your feedback.

---

## COLLEGE ADMIN GUIDANCE

### How to Create an Event:

1. Login with College Admin credentials.
2. Open the Admin Dashboard.
3. Select "Create Event".
4. Fill in:
   - Event Name
   - Type
   - Description
   - Date & Time
   - Venue
   - Registration Deadline
   - Maximum Participants
5. Publish the event.

---

### How to Manage Participants:

1. Open Admin Dashboard.
2. Select an event.
3. View registered participants.
4. Review student details.
5. Approve or reject registrations.

## COMMON QUESTIONS

Q: What is CampusEventHub?
A: CampusEventHub is an inter-college event management platform where students can discover and register for events, and college admins can manage events.

Q: Is CampusEventHub free?
A: Creating an account and browsing events is free. Some events may have participation fees set by organizers.

Q: Can students register for events from other colleges?
A: Yes, CampusEventHub supports multi-college event participation.

Q: How do I check if my registration is approved?
A: Login and check the status in "My Registrations".

Q: How do I become a College Admin?
A: Admin accounts are created after verification by the platform administrators.
